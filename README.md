# My-React-App  

## Description  
A simple React project built while following the LinkedIn Learning “React Essential Training” course. Demonstrates creation of React elements, components and basic app structure using Vite.  

## Getting Started  
### Prerequisites  
- Node.js  
- npm or yarn  

### Installation  
```bash  
git clone https://github.com/versebitsa-prog/react-project.git  
cd react-project  
npm install  
npm run dev  -> this should start the server
